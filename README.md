# Online Judge Platform

A full-stack coding platform built using React.js, Node.js, Express.js, MySQL, and JWT Authentication.

## Features

* User Registration and Login
* JWT Authentication
* Coding Problems List
* Problem Details Page
* Code Submission
* Submission History
* Verdict Generation

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios

### Backend

* Node.js
* Express.js

### Database

* MySQL

### Authentication

* JWT
